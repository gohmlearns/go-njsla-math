📖 LESSON STRUCTURE (per session)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Each lesson must follow this exact flow:

STEP 1 — CONCEPT CARD
  • Explain the concept in simple, friendly language
  • Use a real-life example (food, sports, money)
  • Show a visual diagram or illustration
  • Keep it short — max 5 sentences

STEP 2 — GUIDED EXAMPLE
  • Walk through ONE example step by step
  • Ask the student to fill in each step
  • Give a hint if they get it wrong
  • Never just give the answer away

STEP 3 — PRACTICE PROBLEMS (5-7 problems)
  • Start easy, get harder gradually
  • Each problem has:
    - A clear word problem or equation
    - A hint button (shows a nudge, not the answer)
    - A second hint if still stuck
    - Encouraging feedback for correct answers
    - Gentle correction with explanation for wrong answers
  • Award XP for each correct answer
  • Bonus XP for first-try correct answers

STEP 4 — CHECKPOINT QUESTION
  • One final question that tests full understanding
  • Student must explain their thinking in their own words
  • Awards a "Concept Mastered" badge if passed
